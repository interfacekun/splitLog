#  分割日志工具
## 使用方法
### linux:
#### 1.将需割的日志放入rawLogs文件夹，可一次性放多个
#### 2.打开 终端 cd进入splitLog目录执行：
```Bash  
python splitLog.py
```
#### 3.解压后的日志文件在slicesLogs文件夹下

### windows:
#### 1.将需割的日志放入rawLogs文件夹，可一次性放多个
#### 2.打开 cmd cd进入logTool目录执行：
```Bash  
python splitLog.py
```
#### 3.解压后的日志文件在slicesLogs文件夹下
